#  通过swiper插件实现移动端轮播图

官网：https://swiper.com.cn/

老师是用京东的静态网页来做，我使用携程的静态网页来做

下载一个swiper，这里下载的是swiper-4.5.3.zip

引入css和js
```html
<link rel="stylesheet" href="css/swiper.min.css">
<script src="js/swiper.min.js"></script>
```

参考030-pagination.html写代码

这个好像以前做过

有多个swiper模板，你可以自己试试。
